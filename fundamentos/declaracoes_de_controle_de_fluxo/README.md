# Declarações de Controle de Fluxo

As instruções dentro dos seus arquivos-fonte geralmente são executadas de cima para baixo, na ordem em que aparecem.  

As **declarações de controle de fluxo**, no entanto, interrompem o fluxo de execução ao empregar **decisão**, **laços de repetição** e **ramificações**, permitindo que seu programa execute condicionalmente determinados blocos de código.  

Esta seção descreve:  
- As **declarações de decisão**: `if-then`, `if-then-else`, `switch`  
- As **declarações de laço**: `for`, `while`, `do-while`  
- As **declarações de ramificação**: `break`, `continue`, `return`  

Todas essas são suportadas pela linguagem de programação Java.
