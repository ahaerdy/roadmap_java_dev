# Herança

A herança é um conceito fundamental na programação orientada a objetos, onde uma nova classe (chamada de subclasse ou classe derivada) adquire propriedades e comportamentos de uma classe existente (chamada de superclasse ou classe base). Isso permite a reutilização de código e a criação de relações hierárquicas entre classes, promovendo uma base de código mais organizada e fácil de manter. A subclasse pode estender a superclasse adicionando novos atributos e métodos ou sobrescrevendo os já existentes.
