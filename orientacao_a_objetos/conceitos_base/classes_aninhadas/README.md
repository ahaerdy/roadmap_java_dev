# Classes Aninhadas (Nested Classes)

Classes aninhadas são classes definidas dentro de outra classe. A classe que contém a classe interna é conhecida como classe externa (ou *outer class*). As classes aninhadas podem acessar os membros da classe externa, mesmo que eles sejam declarados como privados. Elas são uma maneira de agrupar logicamente classes que são organizadas ou utilizadas em apenas um lugar, aumentando o encapsulamento e a manutenibilidade.
