# Palavra-chave `static`

A palavra-chave `static` em Java é usada para criar membros (variáveis e métodos) que pertencem à própria classe, em vez de à qualquer instância específica da classe. Isso significa que existe apenas uma cópia de uma variável `static` compartilhada por todos os objetos dessa classe, e você pode acessar membros `static` diretamente usando o nome da classe sem precisar criar um objeto. Métodos `static` só podem acessar variáveis `static` e chamar outros métodos `static`.
