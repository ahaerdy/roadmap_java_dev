# Atributos e Métodos

Atributos são variáveis ​​que armazenam dados sobre um objeto, definindo seu estado ou características. Métodos são funções que definem o comportamento de um objeto, permitindo que ele execute ações ou operações. Juntos, atributos e métodos encapsulam os dados e o comportamento de um objeto dentro de uma classe.
