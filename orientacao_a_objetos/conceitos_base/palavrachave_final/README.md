# Palavra-chave `final`  

A palavra-chave `final` em Java é um modificador não relacionado a acesso, usado para aplicar restrições a uma variável, método ou classe.

Quando aplicada a uma variável, torna o valor da variável constante após a inicialização.

Quando aplicada a um método, impede que o método seja sobrescrito em uma subclasse.

Quando aplicada a uma classe, impede que a classe seja estendida (herdada).
